# Q014. Типы операторов в JavaScript?

> **Источник:** [09. js.md](../09.%20js.md) · **Тема:** JavaScript

---

## Короткий ответ

JavaScript имеет несколько категорий операторов: арифметические (`+`, `-`, `*`, `/`, `%`, `**`), операторы сравнения (`==`, `===`, `<`, `>`), логические (`&&`, `||`, `!`, `??`), операторы присваивания (`=`, `+=`, …), побитовые (`&`, `|`, `^`, `~`, `<<`, `>>`), операторы работы с типами (`typeof`, `instanceof`), тернарный `?:`, унарные (`+x`, `-x`, `++`, `--`), операторы распространения/деструктуризации (`...`) и специальные (`void`, `delete`, `in`, `new`).

---

## Развёрнутый ответ

### Суть и определение

Оператор — синтаксическая конструкция, выполняющая операцию над одним или несколькими операндами и возвращающая результат. Операторы классифицируются по:
- **Арности:** унарный (1 операнд), бинарный (2), тернарный (3).
- **Категории:** арифметические, логические, сравнения и т.д.

### Основные категории операторов

**1. Арифметические:**
```javascript
5 + 3    // 8   (также конкатенация строк)
5 - 3    // 2
5 * 3    // 15
5 / 3    // 1.666...
5 % 3    // 2   (остаток)
2 ** 10  // 1024 (возведение в степень, ES2016)
++x      // префиксный инкремент
x++      // постфиксный инкремент
```

**2. Присваивания:**
```javascript
x = 5
x += 3   // x = x + 3
x -= 3   // x = x - 3
x **= 2  // x = x ** 2
x &&= y  // x = x && y (ES2021, логическое присваивание)
x ||= y  // x = x || y
x ??= y  // x = x ?? y
```

**3. Сравнения:**
```javascript
a == b    // нестрогое (coercion)
a === b   // строгое
a != b    // нестрогое неравенство
a !== b   // строгое неравенство
a < b, a > b, a <= b, a >= b
```

**4. Логические:**
```javascript
a && b   // И — возвращает первый falsy или последний операнд
a || b   // ИЛИ — возвращает первый truthy или последний операнд
!a       // НЕ
a ?? b   // Nullish coalescing — b только если a === null/undefined
```

**5. Побитовые (битовые):**
```javascript
a & b    // AND
a | b    // OR
a ^ b    // XOR
~a       // NOT
a << n   // сдвиг влево
a >> n   // сдвиг вправо со знаком
a >>> n  // беззнаковый сдвиг вправо
```

**6. Типа и принадлежности:**
```javascript
typeof x      // строка с типом
x instanceof Constructor
"key" in obj  // наличие ключа
```

**7. Специальные:**
```javascript
condition ? valueIfTrue : valueIfFalse  // тернарный
delete obj.key     // удаление свойства
void expression    // возвращает undefined
new Constructor()  // создание экземпляра
...arr             // spread
```

### Практика и применение

- **Приоритет операторов (precedence):** определяет порядок вычисления без скобок. `*` выше `+`. `&&` ниже `===`. Лучшая практика — **явные скобки** для неочевидных выражений.
- **Short-circuit evaluation:** `&&` и `||` «ленивы» — правый операнд вычисляется только если нужно. Используется для условного рендеринга в React: `isAuth && <Dashboard />`.
- **Nullish coalescing `??`:** появился в ES2020, решает проблему `||` со значениями `0`, `""`.

### Важные нюансы и краеугольные камни

- Постфиксный `x++` возвращает значение **до** инкремента; префиксный `++x` — **после**.
- `delete` на переменную в strict mode бросает `SyntaxError`; работает только на свойствах объекта.
- `void` всегда возвращает `undefined`; используется в `void 0` как гарантированный `undefined`.
- Побитовые операторы конвертируют число в 32-битный integer — потеря точности для больших чисел.

### Примеры

```javascript
// Short-circuit
const user = null;
const name = user && user.name;      // null (не бросает ошибку)
const display = name || 'Anonymous'; // 'Anonymous'
const greeting = name ?? 'Guest';    // 'Guest' (только null/undefined)

// Логические присваивания (ES2021)
let config = {};
config.timeout ??= 3000;  // присвоит 3000, если timeout был null/undefined
config.retries ||= 3;     // присвоит 3, если retries falsy

// Тернарный оператор vs if
const status = isActive ? 'active' : 'inactive';

// void для ссылок без навигации
<a href="javascript:void(0)" onClick={handler}>Click</a>
```

## На что обратить внимание на собеседовании

### Каверзные follow-up (что спросят дальше)

- **Чем `??` отличается от `||`?** — `||` срабатывает на любое falsy (0, "", false); `??` только на null/undefined.
- **Что вернёт `x++` и `++x`?** — `x++` — старое значение, `++x` — новое.
- **Как работает short-circuit в `&&`?** — Возвращает первый falsy операнд или последний, если все truthy.

### Красные флаги (чего не говорить)

- «`||` и `??` — одно и то же» — принципиально разные семантики для `0`, `""`, `false`.
- «Побитовые операторы редко нужны во фронте» — используются в масках прав, флагах, хэшах, графике (Canvas).
- «`delete` удаляет переменную» — только свойства объекта; переменные не удаляются.

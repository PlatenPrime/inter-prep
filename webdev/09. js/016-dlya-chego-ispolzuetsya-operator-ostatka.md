# Q016. Для чего используется оператор остатка (`%`)?

> **Источник:** [09. js.md](../09.%20js.md) · **Тема:** JavaScript

---

## Короткий ответ

Оператор `%` возвращает остаток от целочисленного деления двух операндов. Знак результата совпадает со знаком **делимого** (левого операнда). Используется для циклических диапазонов, определения чётности/нечётности, работы с временными метками и любых задач, требующих «оборачивания» числового диапазона.

---

## Развёрнутый ответ

### Суть и определение

`a % b` возвращает `r`, такое что `a = q * b + r`, где `q` — целая часть `a / b`, а `|r| < |b|`. В отличие от математического modulo, знак остатка в JS зависит от знака **делимого**:

```javascript
7 % 3   //  1  (положительное делимое)
-7 % 3  // -1  (отрицательное делимое → остаток отрицательный)
7 % -3  //  1  (знак делимого — 7 — положительный)
```

### Практика и применение

```javascript
// Чётность / нечётность
const isEven = n => n % 2 === 0;
const isOdd  = n => n % 2 !== 0;

// Циклический индекс (карусель, round-robin)
function next(current, total) {
  return (current + 1) % total;
}
next(4, 5); // 0 — оборачивается

// Проверка делимости
const isDivisibleBy3 = n => n % 3 === 0;

// Извлечение последних N цифр числа
12345 % 100; // 45

// Время: минуты из секунд
function formatSeconds(totalSeconds) {
  const minutes = Math.floor(totalSeconds / 60);
  const seconds = totalSeconds % 60;
  return `${minutes}:${String(seconds).padStart(2, '0')}`;
}
formatSeconds(125); // "2:05"
```

### Важные нюансы и краеугольные камни

- Для истинного математического modulo (всегда неотрицательный результат) нужна корректировка: `((n % m) + m) % m`.
- `% 2` на отрицательных числах: `-3 % 2 === -1`, не `1` — ловушка при проверке чётности отрицательных.
- `NaN % anything` → `NaN`, `anything % 0` → `NaN`, `Infinity % anything` → `NaN`.
- Оператор не работает с `BigInt` и `Number` одновременно: `5n % 2` → `TypeError`.

### Примеры

```javascript
// Математический modulo (всегда >= 0)
function mod(a, m) {
  return ((a % m) + m) % m;
}
mod(-7, 3);  // 2  (а не -1)
mod(7, 3);   // 1

// Распределение по группам (шардинг)
function getShardIndex(userId, totalShards) {
  return userId % totalShards;
}

// Цикличная анимация (текущий кадр из N)
const frame = (Date.now() / 100 | 0) % totalFrames;
```

## На что обратить внимание на собеседовании

### Каверзные follow-up (что спросят дальше)

- **Почему `-7 % 3 === -1`, а не `2`?** — JS использует remainder, не modulo; знак — от делимого.
- **Как реализовать настоящий modulo?** — `((a % m) + m) % m`.
- **Что вернёт `5 % 0`?** — `NaN`.

### Красные флаги (чего не говорить)

- «`%` — это оператор деления по модулю» — технически это remainder, modulo — математически другое при отрицательных числах.
